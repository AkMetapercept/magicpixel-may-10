# Edit a Qualification Criteria

You can edit the Qualification Criteria details of a particular criteria that is already created.

To edit the Qualification Criteria in MagicPixel, perform the following steps:

1.	Select the Qualification Criteria that you want to edit.
2.	From the **Actions** drop-down menu on the top right corner, select **Edit**.

![editaqualificationcriteria](/images/qualificationcriteriaimages/edit.png)

3.	You can edit the details under Info section and Data Elements section and click **Save** to save the changes.
