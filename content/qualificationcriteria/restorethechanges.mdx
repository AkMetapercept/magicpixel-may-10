# Restore the Changes

Once deleted, you can revert the changes using the Restore option.

To restore the changes, perform the following steps:

1. Navigate to the Qualification Criteria screen to view the deleted Qualification Criteria.

2. Select the Qualification Criteria and click **Restore** under the **Actions** drop-down on the top right corner.

![restorethechanges](/images/qualificationcriteriaimages/restore.png)

3. Click **Yes** in the Restore confirmatory dialog box that appears.

![restorethechanges](/images/qualificationcriteriaimages/restoreconfirmatory.png)

**Step Result**: The Qualification Criteria is restored successfully.
