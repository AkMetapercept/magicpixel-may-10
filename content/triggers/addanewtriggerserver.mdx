# Add a New Trigger (Server-side)

For Server-side configuration, all the information is fetched from the server.

To add a new Trigger on Ingest Labs, perform the following steps:

1.	From the left navigation panel, select **Triggers**.

![addanewtriggerserver](/images/triggersimages/triggersserver.png)

2.	Select **Add Trigger -> For Server** on the top-right corner of the screen.

![addanewtriggerserver](/images/triggersimages/triggersforserver.png)

3.	Enter the Trigger Name in the **Name** field under Info section.

4.	Select the Data Element from the Data Element section.

5.	Click **Save** to save the trigger details.
