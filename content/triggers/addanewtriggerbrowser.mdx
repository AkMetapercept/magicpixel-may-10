# Add a New Trigger (Browser/Client-side)

For Client-side or Browser-side configuration, all the information is fetched from the webpage or browser.
To add a new Provider on MagicPixel, perform the following steps:
1.	From the left navigation panel, select **Triggers**.

![addanewtriggerbrowser](/images/triggersimages/triggersbrowser.png)

2.	Select Add **Trigger -> For Browser** on the top-right corner of the screen.

![addanewtriggerbrowser](/images/triggersimages/triggersforbrowser.png)

3.	Enter the Trigger Name in the **Name** field under Info section.
4.	Select the Data Element from the Data Element section.
5.	Click **Save** to save the trigger details.
