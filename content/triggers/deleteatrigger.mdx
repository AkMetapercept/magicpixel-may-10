# Delete a Trigger

You can delete a particular Trigger.

To delete a Trigger in MagicPixel, perform the following steps:
1.	Select the Trigger that you want to delete.
2.	From the Actions drop-down menu on the top right corner, select **Delete**.

![deleteatrigger](/images/triggersimages/delete.png)

3.	Click **Yes** on the confirmatory message that appears.

![deleteatrigger](/images/triggersimages/deleteconfirmatory.png)

**Step Result**: The Trigger is deleted successfully.
