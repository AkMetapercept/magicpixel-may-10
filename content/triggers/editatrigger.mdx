# Edit a Trigger

You can edit the Trigger details of a particular Trigger that is already created.

To edit the Trigger in Ingest Labs, perform the following steps:

1.	Select the Trigger that you want to edit.

2.	From the **Actions** drop-down menu on the top right corner, select **Edit**.

![editatrigger](/images/triggersimages/edit.png)

3.	You can edit the details under Info section and Data Elements section and click **Save** to save the changes.
