# Restore the changes

Once deleted, you can revert the changes using the **Restore** option. 

To restore the changes, perform the following steps:

1. Select the Trigger and click **Restore** under the **Actions** drop-down on the top right corner.

![restorethechanges](/images/triggersimages/restore.png)

2. Click **Yes** in the Restore confirmatory dialog box that appears.

![restorethechanges](/images/triggersimages/restoretrigger.png)

**Step Result**: The Trigger is restored successfully.
