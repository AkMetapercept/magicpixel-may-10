# Restore the Changes

Once deleted, you can revert the changes using the **Restore** option.

To restore the changes, perform the following steps:

1. Navigate to the Tags screen to view the deleted tag.

2. Select the tag and click **Restore** under the **Actions** drop-down on the top right corner.

3. Click **Yes** in the Restore confirmatory dialog box that appears.

![restorethechanges](/images/tagsimages/restoretag.png)

**Step Result**: The tag is restored successfully.
