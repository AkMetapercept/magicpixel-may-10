# Edit a Tag
You can edit the Tag details of a particular tag.

To edit the tag in Ingest Labs, perform the following steps:

1.	Select the tag that you want to edit.

2.	From the **Actions** drop-down menu on the top right corner, select **Edit**.

![editatag](/images/tagsimages/editatag.png)

**Note**: For any tag, you cannot edit the Provider Attributes but you can update/edit the Triggers, Qualification Criteria, Tag Info, and Tag Attributes.

3.	Edit the tag details and click **Save** on the top-right to save the changes.
