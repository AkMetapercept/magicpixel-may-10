# Delete a Tag
You can delete a particular tag.

To delete a tag in MagicPixel, perform the following steps:
1.	Select the tag that you want to delete.
2.	From the **Actions** drop-down menu on the top right corner, select **Delete**.

![deleteatag](/images/tagsimages/deleteatag.png)

3.	Click **Yes** on the confirmatory message that appears.

![deleteatag](/images/tagsimages/deletetagconfirmation.png)

**Step Result**: The tag is deleted successfully.
