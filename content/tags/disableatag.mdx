# Disable a Tag

To disable the tag in Ingest Labs, perform the following steps:

1.	Select the tag that you want to disable.

2.	From the **Actions** drop-down menu on the top right corner, select **Disable**.

![disableatag](/images/tagsimages/disableatag.png)

**Note**: You have to select at least one qualification criterion to disable a tag. The tag is disabled successfully.
