# Edit a Data Element

You can edit the details of a particular Data Element.

To edit the Data Element in Ingest Labs, perform the following steps:

1.	Select the Data Element you want to edit.

2.	From the **Actions** drop-down menu on the top right corner, select **Edit**.

![editadataelement](/images/dataelementsimages/editadataelement.png)

3. Edit the Data Element details and click **Save** on the top-right to save the changes.
