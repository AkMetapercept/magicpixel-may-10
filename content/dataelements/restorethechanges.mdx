# Restore the Changes

Once deleted, you can revert the changes using the **Restore** option. 

To restore the changes, perform the following steps:

1. Navigate to the Data Elements screen to view the deleted Data Element.

2. Select the Data Element and click **Restore** under the **Actions** drop-down on the top right corner.

![restorethechanges](/images/dataelementsimages/restore.png)

3.	Click **Yes** in the Restore confirmatory dialog box that appears.

![restorethechanges](/images/dataelementsimages/restoreconfirmatory.png)

**Step Result**: The Data Element is restored successfully.
