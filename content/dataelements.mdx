# Data Elements

Data Elements are the core components you define while creating the tags. Using Data Elements, you can retrieve data from the website or mobile application through the URL. You can use the Default Data Element templates for multiple projects. Using the Data Element template saves time spent in creating a new data element every time. You can create any number of Data elements for a particular Data type and define the elements in the tags as per client requirements.
Some of the Data types include Server Header, Server Cookie, Ingest Labs Data Layer, URL Parameter, Custom JavaScript, Custom event, and so on.

**Note**: Using Administrator privileges, you can add new default Data Elements using the Settings option available on the left navigation panel. **Settings -> Templates -> Available Templates**. 

You can navigate to any of the Tag and add new Data Elements from the Data Elements section in the **Apply Template** screen.
