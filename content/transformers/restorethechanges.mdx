# Restore the Changes

Once deleted, you can revert the changes using the **Restore** option. 

To restore the changes, perform the following steps:

1. Navigate to the Transformers screen to view the deleted transfomer.

1. Select the Transformer and click **Restore** under the **Actions** drop-down on the top right corner.

![restorethechanges](/images/transformersimages/restore.png)

2. Click **Yes** in the Restore confirmatory dialog box that appears.

![restorethechanges](/images/transformersimages/restoreconfirmatory.png)

**Step Result**: The Transformer is restored successfully.
