# Add a New Transformer (For Server-side)

For Server-side configuration, all the information is fetched from the server.

To add a new Transformer on Ingest Labs, perform the following steps:

1.	In the left navigation panel, select **Transformers**.

![addanewtransformerserver](/images/transformersimages/transformersserver.png)

2.	Select **Add Transformer -> For Server** on the top-right corner of the screen.

![addanewtransformerserver](/images/transformersimages/addtransformersforserver.png)

3.	Enter the Function Name in the **Function Name** field under Info section.

**Note**: The Function name must always start with an alphabet, and can only contain alphabets, numbers and underscores.

4.	Enter the Description in the **Description** field.

![addanewtransformerserver](/images/transformersimages/newtransformersserver.png)

5.	Click Add Arguments in the **Function Arguments** section.

![addanewtransformerserver](/images/transformersimages/addargumentserver.png)

6.	Enter the Argument name and select the Data Element under the **Value** section.

7.	Select the check boxes for Mandatory and URL Encode, if necessary.

![addanewtransformerserver](/images/transformersimages/functionargumentsserver.png)

8.	To delete the Argument, click the **Delete** icon on the right-hand corner of the screen.

9.	To access the Argument, enter the code in the Code section. You can also test the function using the **Test Function** tab on the right-hand corner of the screen.

![addanewtransformerserver](/images/transformersimages/codeserver.png)

10.	Click **Save** to save the Transformers details.
