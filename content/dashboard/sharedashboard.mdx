# Share Dashboard

You can share the dashboard with multiple users. 

Perform the following steps to share the Dashboard with other users:

1.	Select the **Share Dashboard** option from the newly created dashboard option.

![sharedashboard](/images/dashboardimages/sharedashboard.png)

2.	Enter the e-mail ID in the **Add New User** field and click **Add User**.

**Note:** A notification is triggered to the users about the dashboard details.

3.	Click the **Delete** icon against the user's e-mail ID to revoke the option.
