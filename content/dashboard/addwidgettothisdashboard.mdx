# Add Widget to Dashboard

From the new dashboard that is created, you can add new widgets.

Select **Add Widget to this Dashboard** option from the newly created dashboard.

![addwidgettodashboard](/images/dashboardimages/addwidgetstodashboard.png)

To add a new widget in the dashboard, follow the procedure "Add Widgets in the Dashboard".
