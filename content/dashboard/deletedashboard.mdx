# Delete Dashboard

You can delete the newly created dashboard using this option.

To delete the dashboard, perform the following steps:

1.	Select **Delete Dashboard** from the newly created dashboard drop-down menu:

![deletedashboard](/images/dashboardimages/deletedashboard.png)

2.	In the confirmatory dialog box that appears, click **Yes**.

![deletedashboard](/images/dashboardimages/deletesuccess.png)

**Step Result:** The Dashboard is deleted successfully along with the widgets created.
