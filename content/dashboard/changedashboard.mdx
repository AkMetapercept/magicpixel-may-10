# Change Dashboard

You can change the Dashboard and revert to the Default dashboard.

Perform the following steps to change the Dashboard.

1. From the new Dashboard, select the **Change Dashboard** option.

2. Click the **Project's Default Dashboard** option on the Dashboard screen.

![changedashboard](/images/dashboardimages/changedashboard.png)

**Step Result:** The default dashboard appears on the screen.

Or

3. While viewing the Default Dashboard, you can select and navigate to the specific Dashboard to display.
