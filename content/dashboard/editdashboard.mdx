# Edit Dashboard

You can edit the Dashboard details of the newly created dashboard.

To edit the Dashboard details, perform the following steps:

1. Select **Edit Dashboard** from the newly created dashboard drop-down menu.

![editdashboard](/images/dashboardimages/editdashboard.png)

2. Edit the details in the **Update Dashboard** and click **Save**.

**Step Result:** The dashboard details are updated.
